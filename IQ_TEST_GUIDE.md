# 🧠 IQ 인지능력 테스트 사용 설명서

이 문서는 IQ 인지능력 테스트 애플리케이션의 상세 사용법과 평가 기준을 설명합니다.

## 1. 테스트 개요
이 테스트는 단순한 지식 테스트가 아닌, **순수 인지 능력**을 측정하기 위해 설계되었습니다. 총 25문항으로 구성되며, 5가지 핵심 인지 영역을 종합적으로 평가합니다.

## 2. 시작하기
1. **접속**: 웹사이트에 접속하면 메인 대시보드가 나타납니다.
2. **영역 선택**: 5가지 테스트 영역 중 하나를 선택합니다. (패턴, 수리, 공간, 논리, 언어)
3. **테스트 시작**: '테스트 시작' 버튼을 누르면 해당 영역의 10문제가 출제됩니다.

## 3. 멘사 도전 (Mensa Challenge) 🏆
이 애플리케이션의 핵심 기능입니다.

*   **자격 획득**: 일반 테스트에서 **IQ 120 이상**을 기록하면 '멘사 도전 자격증'과 **고유 코드**가 발급됩니다.
*   **도전 방법**: 메인 화면 하단의 '멘사 도전 테스트' 입력창에 코드를 입력하면 잠금된 멘사 테스트가 열립니다.
*   **난이도**: 멘사 테스트는 훨씬 높은 난이도의 20문항으로 구성되어 있습니다.

## 3. 화면 구성 및 조작법
*   **상단 정보**:
    *   **진행 바**: 전체 25문제 중 현재 위치를 보여줍니다.
    *   **타이머**: 문제당 **60초**의 제한 시간이 있습니다. 시간이 다 되면 자동으로 다음 문제로 넘어갑니다.
    *   **카테고리**: 현재 문제의 유형(예: 패턴 인식)이 표시됩니다.
*   **문제 영역**: 중앙에 문제 이미지나 텍스트, 수열 등이 표시됩니다.
*   **선택지**: 하단에 있는 보기 중 정답이라고 생각되는 것을 클릭/터치합니다.
*   **네비게이션**:
    *   **건너뛰기**: 모르는 문제는 건너뛸 수 있습니다 (오답 처리됨).
    *   **다음 문제**: 답을 선택하면 버튼이 활성화됩니다.

## 4. 평가 영역 (5가지)
각 영역별로 5문제씩 출제됩니다.

1.  **패턴 인식 (Pattern Recognition)**
    *   도형이나 그림의 규칙성을 찾아 빈칸을 채우는 능력입니다.
    *   *팁: 색상 변화, 회전, 개수 증가 등의 규칙을 찾으세요.*
2.  **수열 추론 (Numerical Reasoning)**
    *   나열된 숫자의 규칙을 파악하여 다음 숫자를 예측합니다.
    *   *팁: 등차, 등비, 피보나치 등 수학적 패턴을 고려하세요.*
3.  **공간 추론 (Spatial Reasoning)**
    *   도형의 회전, 전개도, 단면 등을 머릿속으로 시뮬레이션하는 능력입니다.
    *   *팁: 도형을 마음속으로 돌려보거나 접어보세요.*
4.  **논리 추론 (Logical Reasoning)**
    *   주어진 명제나 조건으로부터 타당한 결론을 도출합니다.
    *   *팁: '만약 ~라면' 가정을 세우고 모순이 없는지 확인하세요.*
5.  **언어 유추 (Verbal Analogies)**
    *   단어 간의 관계(유의어, 반의어, 포함 관계 등)를 파악합니다.
    *   *팁: 'A는 B의 ~이다'라는 문장을 만들어 관계를 대입해보세요.*

## 5. 점수 산정 방식 (고급 알고리즘)
단순히 맞춘 개수로만 평가하지 않습니다.

*   **난이도 가중치**: 쉬운 문제보다 어려운 문제(난이도 3)를 맞출수록 점수가 높습니다.
*   **시간 보너스**: 문제를 빨리 맞출수록 추가 점수(최대 20%)가 부여됩니다.
    *   *전략: 정확도가 최우선이지만, 확신이 든다면 빠르게 선택하는 것이 유리합니다.*

## 6. 결과 해석
테스트 종료 후 즉시 결과가 분석됩니다.

*   **IQ 점수**: 표준 정규분포(평균 100, 표준편차 15) 기준의 추정치입니다.
*   **백분위**: 전체 인구 중 본인의 위치를 나타냅니다 (예: 상위 1%).
*   **영역별 분석**: 5가지 영역 중 강점과 약점을 그래프로 보여줍니다.
