# 🧠 IQ 인지능력 테스트 (IQ Cognitive Ability Test)

과학적 문제 설계를 기반으로 한 종합 인지능력 평가 웹 애플리케이션입니다.

## 🌟 주요 기능

## 🌟 주요 기능

*   **5가지 인지 영역 선택**:
    *   원하는 영역을 선택하여 집중적으로 테스트할 수 있습니다.
    *   패턴 인식, 수열 추론, 공간 지각, 논리 추론, 언어 유추
*   **🏆 멘사 도전 모드**:
    *   일반 테스트에서 **IQ 120 이상** 달성 시 멘사 도전 자격(코드)이 부여됩니다.
    *   더 높은 난이도의 문제로 구성된 멘사 테스트에 도전할 수 있습니다.
*   **정밀한 분석**: 영역별 상세 분석 및 백분위 제공
*   **다크 모드 & 반응형**: 몰입감 있는 디자인과 모바일 최적화

## 🚀 실행 방법

1. `index.html` 파일을 엽니다.
2. 메인 화면에서 원하는 테스트 영역을 선택합니다.
3. 테스트 완료 후 자격을 획득하면 멘사 테스트에 도전해보세요!

## 🛠️ 기술 스택

*   CSS3 (Flexbox, Grid)
*   Vanilla JavaScript (ES6+)
