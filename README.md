# 🧠 IQ 인지능력 테스트 (IQ Cognitive Ability Test)

과학적 문제 설계를 기반으로 한 종합 인지능력 평가 웹 애플리케이션입니다.

## 🌟 주요 기능

*   **5가지 인지 영역 평가**:
    *   패턴 인식 (Pattern Recognition)
    *   수열 추론 (Numerical Reasoning)
    *   공간 추론 (Spatial Reasoning)
    *   논리 추론 (Logical Reasoning)
    *   언어 유추 (Verbal Analogies)
*   **정밀한 점수 산정**: 문제 난이도와 풀이 시간을 고려한 가중치 점수 시스템 적용
*   **반응형 디자인**: PC와 모바일 모두에서 최적화된 UI 제공
*   **다크 모드**: 눈이 편안한 다크 테마 적용

## 🚀 실행 방법

이 프로젝트는 정적 웹페이지로 구성되어 있어 별도의 설치가 필요 없습니다.

1. `index.html` 파일을 웹 브라우저로 엽니다.
2. 또는 [GitHub Pages 링크](https://your-username.github.io/iq-test)를 통해 접속합니다. (배포 후 링크 수정 필요)
